# powerbi-customer-segmentation
End-to-end Power BI customer segmentation analysis using the Online Retail II dataset — including data cleaning, star schema modeling and insights.

Using Kaggle's Online Retail II dataset which contains online retail transaction data set of two years, my goal was to identify different customer segments based on purchasing behavior using RFM (Recency, Frequency, Monetary) analysis.

The key questions the report looks to answer are:

	- Who are the most valuable customers?

	- Which customers are at risk of churning?

	- How much revenue comes from each segment?

	- How do purchasing patterns differ between segments?
